# Spamdetection-Project.ipynb
📧 Spam Detection System using Machine Learning and NLP. This project classifies SMS messages as spam or ham using TF-IDF vectorization and Multinomial Naive Bayes. It includes preprocessing, training, evaluation, and real-time prediction of new messages.
🚀 Features
Text preprocessing and cleaning
Feature extraction using TF-IDF
Model training using Multinomial Naive Bayes
Model evaluation using accuracy, confusion matrix, and classification report
Real-time prediction on new messages
🛠️ Technologies Used
Python
Pandas
Scikit-learn
NLP (TF-IDF Vectorization)
📊 Outcome

The model achieves good accuracy in identifying spam messages and demonstrates how machine learning can be applied to real-world text classification problems.
